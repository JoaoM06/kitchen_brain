# `backend/app/db/models`

Descrição rápida das entidades:

| Arquivo | Função |
| --- | --- |
| `user.py` | Usuários/autenticação. |
| `device.py` | Dispositivos registrados. |
| `lgpd.py` | Consentimentos LGPD. |
| `market.py` | Mercados usados no mapa. |
| `media.py` | Mídias (foto/vídeo) do Recipe Hub. |
| `product.py` | Produtos e códigos de barras. |
| `recipe.py` | Receitas e cardápios. |
| `settings.py` | Configurações do usuário. |
| `shopping.py` | Listas/compras. |
| `storage.py` | Armazenamento genérico/attachments.

Adicione novos models aqui e lembre-se de ajustar migrações.
