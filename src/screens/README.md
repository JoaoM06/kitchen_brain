# `mobile/src/screens`

Mapeamento das telas registradas:

| Tela | Função |
| --- | --- |
| `WelcomeScreen.jsx` | Landing inicial para levar a login/onboarding. |
| `LoginScreen.jsx` / `SignupScreen.jsx` | Fluxos de autenticação. |
| `OnboardingScreen.jsx` | Guia passo a passo para novos usuários. |
| `RecipesScreen.jsx` | Descoberta de receitas com CTA para o Hub. |
| `RecipeDetailScreen.jsx` | Detalhes completos da receita. |
| `RecipeHubScreen.jsx` | Feed social com filtros, curtidas, salvamentos e avaliações. |
| `RecipeHubCreateScreen.jsx` | Publicação de receitas com upload de foto/vídeo. |
| `StockScreen.jsx` | Visão do estoque + ações rápidas. |
| `AddItemOptionsScreen.jsx`, `ManualAddScreen.jsx`, `BarcodeScannerScreen.jsx` | Diferentes formas de adicionar itens. |
| `CardapioBotScreen.jsx`, `MenuViewScreen.jsx` | Geração e visualização de cardápios. |
| `MarketMapScreen.native/web.jsx` | Mapa de mercados (nativo vs web). |
| `PermissionsScreen.jsx` | Permissões LGPD e integrações (push, câmera, etc.). |
| `VoiceRecScreen.jsx` | Gravador e transcritor. |
| `ProfileScreen.jsx` | Perfil com abas (Receitas salvas, Postagens, Cardápios). |
| `ConfigsScreen.jsx`, `HelpScreen.jsx`, `AboutScreen.jsx`, `AccessibilityScreen.jsx` | Configurações diversas, ajuda e acessibilidade. |

Use esta lista como índice rápido de rotas.
