# `mobile/src/components`

Biblioteca de componentes reutilizáveis.

| Componente | Descrição |
| --- | --- |
| `DefaultButton.jsx` | Botão padrão (cheio e outline) com ripple acessível. |
| `DefaultInput.jsx` | Campo de texto com espaçamento consistente. |
| `PasswordInput.jsx` | Variante com toggle de visibilidade. |
| `FooterNav.jsx` | Barra inferior com tabs para Receitas, Hub, Estoque e Perfil. |
| `SafeScreen.jsx` | Wrapper usando `SafeAreaView`. |
| `SuccessModal.jsx` | Modal genérico de sucesso/confirm.
