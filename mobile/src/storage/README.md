# `mobile/src/storage`

Persistência local usando AsyncStorage (ou `localStorage` no web).

| Arquivo | Descrição |
| --- | --- |
| `savedMenus.js` | Salva cardápios gerados (PDF, datas, restrições). |
| `recipeHub.js` | Guarda posts criados no dispositivo e reações (likes/saves/notas) aos posts curados.

Helpers retornam listas atualizadas para facilitar o binding em hooks/telas.
